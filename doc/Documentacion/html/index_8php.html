run ( )

Main router (single entry-point for all requests) of the MVC implementation.

This router will create an instance of the corresponding controller, based on the "controller" parameter and call the corresponding method, based on the "action" parameter.

The rest of GET or POST parameters should be handled by the controller itself.

Parameters:

  • controller: The controller name (via HTTP GET)
  • action: The name inside the controller (via HTTP GET)
Devuelve
void
